# API Set UP

The API is Built using Laravel
The Laravel server requirements

- PHP >= 7.1.3
- BCMath PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

[Readmore at Laravel](https://laravel.com/docs/5.8)
